# Responsive Web Design Templates

## Overview

This project aims to showcase the languages used in web page development: HTML (Hypertext Markup Language) for content and CSS (Cascading Style Sheets) for design. It introduces modern techniques, including CSS variables and best practices for accessibility, to create a test site. The web pages are designed to adapt to various screen sizes, providing examples of a photo gallery using Flexbox and a magazine article layout using CSS Grid.

## Features

Before running this application, please ensure that you have the following prerequisites:

- Java Development Kit (JDK)
- Maven (for building the project)
